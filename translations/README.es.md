## 🔧 Instalación Técnica

```bash
# Clonar el repositorio
git clone https://github.com/vram-ai/sigmai.git

# Instalar dependencias
pnpm install

# Iniciar el servidor de desarrollo
pnpm start

# Configuración
cp .env.example .env

Editar .env con tus claves

# Iniciar en modo desarrollo
pnpm dev
```
